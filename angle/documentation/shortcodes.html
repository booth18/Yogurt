Row

A Horizontal section to add content to.

Full Example

[vc_row title="" subtitle="" title_size="normal" title_weight="regular" title_align="center" title_underline="no-underline" swatch="swatch-white-red" top_decoration="none" bottom_decoration="none" height="normal" width="padded" class="" id="" background_image="" background_video="" overlay_colour="#000000" overlay_opacity="0" overlay_grid="off" background_image_size="cover" background_image_repeat="no-repeat" background_image_attachment="scroll"] content goes here [/vc_row]

Options

Name Description Default
title Main Title text
subtitle Smaller subtitle to be shown under the main title text.
title_size Choose size of the font to use in your header
options : normal, super, hyper 		normal
title_weight Choose weight of the font to use in the title
options : regular, black, bold, light, hairline 		regular
title_align Align the text shown in the header left, right or center.
options : center, left, right 		center
title_underline Adds an underline effect below the title & subtitle.
options : underline, no-underline 		no-underline
swatch Choose a color swatch for the section swatch-white-red
top_decoration Choose a style to use as the top decoration.
options : none, arrow, circle, clouds, cross, curtain, left-slope, right-slope, lines, rounded, rounded-inverted, triangle, triangle-inverted, zig-zag, zipper 		none
bottom_decoration Choose a style to use as the bottom decoration.
options : none, arrow, circle, clouds, cross, curtain, left-slope, right-slope, lines, rounded, rounded-inverted, triangle, triangle-inverted, zig-zag, zipper 		none
content
height Choose the amount of padding added to the height of this section
options : normal, short, tiny, nopadding 		normal
width Choose between padded-non padded section
options : padded, no-padded 		padded
class Add an optional class to the section
id Add an optional id to the section
background_image Choose an image to use for this rows background.
background_video Enter url of a video to use for this rows background.
overlay_colour Set the colour of the video & image overlay #000000
overlay_opacity Set the opacity of the video & image overlay 0
overlay_grid Adds an overlay pattern image
options : off, on 		off
background_image_size Set how the image will fit into the section
options : cover, auto 		cover
background_image_repeat Set how the image will be repeated
options : no-repeat, repeat-x, repeat-y, repeat 		no-repeat
background_image_attachment Set the way the background scrolls with the page. Scroll = normal Fixed = Parallax effect.
options : scroll, fixed 		scroll
